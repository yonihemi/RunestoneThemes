# Runestone Themes
Converted themes from https://colorsublime.github.io/

## Installing the themes
- Either one by one, in Runestone -> Settings -> Editor -> Light/Dark theme -> Import Theme; or
- Copy files to `iCloud Drive/Runestone/Themes`.  Runestone watches this folder and automatically picks up new themes.

## Running the conversion
```sh
$ git submodule update --init --recursive
$ swift run
```
